## What is Polygon Animals?

Polygon Animals is a React-based interactive web application that brings awareness to endangered marine species in Thai waters. It combines low-poly geometric art, immersive parallax scrolling, and 3D underwater environments to create a visually stunning educational experience.

Each animal is rendered as a mosaic of colored polygon shards — when assembled, they form a recognizable silhouette. The visual metaphor is intentional: just as each polygon shard is essential to forming the complete image, each species is essential to maintaining ocean ecosystem balance.

## Key features

  <Card title="Low-poly art rendering" icon="triangle">

  </Card>

## Technology stack

| Technology | Purpose |
|-----------|---------|
| React 19 | UI framework and component system |
| Vite | Build tool and development server |
| Framer Motion | Scroll-driven animations and transitions |
| Three.js / @react-three/fiber | 3D underwater environment rendering |
| @react-three/drei | Three.js helper components |
| React Router DOM | Client-side routing between pages |
| CSS clip-path | Polygon shard rendering system |

## App routes

The application uses React Router for navigation between five main views:

| Route | Page | Description |
|-------|------|-------------|
| `/stages` | Stages | Entry point — four-panel stage selector with intro animation |
| `/landing` | Landing | Scrolling underwater showcase of all marine animals |
| `/detail?id=<animalId>` | Detail | Animated polygon assembly with conservation info tabs |
| `/polygon` | Polygon | Horizontal parallax storytelling about low-poly art |
| `/order` | Orderer | Animal ordering interface |

## Conservation mission

The project was created to raise awareness about marine biodiversity in Thailand. The featured species include animals listed as endangered or critically endangered under Thai wildlife protection laws and international conventions like CITES Appendix I.

The use of polygon art carries a deeper message: if any single piece is missing, the whole image falls apart — just as the loss of any species disrupts the entire ecosystem.
